# metamaterial

私は東北大学金森研究室でテラヘルツ領域における透過型メタレンズの設計、製作をしています。

このコードにあるgds.pyは、
_焦点距離f(μm)_, _波長λ(μm)_, _ウエハの厚みH(μm)_, _使うSiの屈折率silicon_ref_, _中心からの最大距離max_r(mm)_, _ウエハの一辺の長さsize(μm)_, _sizeとウエハの中で使用する部分との差adjustment(μm)_
を入力すると、これらのパラメータから設計されるメタレンズのマスクファイルがgds2形式で出力されるものである。

もし、dxf形式に変換したい場合はLinkCADを使えば10秒で変換が可能。
